# License Plate Spotter Game

A modern, lightweight web-based game for tracking license plates spotted from different states and provinces across the USA, Canada, and Mexico.

## Features

- **Modern UI**: Clean, responsive design with glassmorphism effects
- **Simple Toggle**: Tap any license plate to mark as spotted, tap again to unmark
- **Visual Feedback**: Green checkmarks show which plates you've found
- **Multi-Country Support**: Track plates from USA (50 states + DC), Canada (13 provinces/territories), and Mexico (32 states)
- **Quick Jump Selector**: Dropdown menu to quickly jump to any state or province
- **Save/Load Game**: Export your progress as JSON files and import them later
- **Local Storage**: Automatically saves your progress in the browser
- **New Game**: Reset all progress to start fresh
- **Progress Tracking**: See total spotted, total possible, and completion percentage
- **PWA Support**: Install as an app on iOS and Android devices

## How to Use

1. **Open the Game**: Simply open `index.html` in any modern web browser
2. **Mark a Plate as Spotted**: Tap any license plate card to mark it as found (shows a green checkmark âœ“)
3. **Remove a Spotted Plate**: Click directly on the green checkmark to remove it (prevents accidental removal)
4. **Switch Countries**: Use the horizontal tabs to switch between USA, Canada, and Mexico
5. **Jump to Plate**: Use the dropdown menu to quickly scroll to any specific state or province (auto-clears after selection)
6. **Save Progress**: Click "Save Game" to download your progress as a JSON file
7. **Load Progress**: Click "Load Game" to import a previously saved game file
8. **Start Over**: Click "New Game" to reset all progress

## Game Controls

- **Save Game**: Downloads your current progress as a JSON file
- **Load Game**: Allows you to upload a previously saved game file
- **New Game**: Resets all progress (with confirmation)
- **Search**: Filter license plates by state/province name or abbreviation

## Technical Details

- **Pure Web Technologies**: Built with HTML5, CSS3, and vanilla JavaScript
- **No Dependencies**: Runs entirely in the browser without external libraries
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Local Storage**: Automatically saves progress between sessions
- **File Export/Import**: Save and load game states as JSON files

## Installation

1. Download all files (`index.html`, `styles.css`, `script.js`)
2. Place them in the same folder
3. Open `index.html` in your web browser
4. Start spotting license plates!

Enjoy your license plate spotting adventures!
